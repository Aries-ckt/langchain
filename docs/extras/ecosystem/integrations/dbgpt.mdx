# DB-GPT

>[DB-GPT](https://github.com/csunny/DB-GPT) is an experimental open-source project that uses localized GPT large models to interact with your data and environment. With this solution, you can be assured that there is no risk of data leakage, and your data is 100% private and secure.


## Installation and Setup

If you want to get started with db-gpt, follow the [quick start](https://db-gpt.readthedocs.io/en/latest/getting_started/getting_started.html).
